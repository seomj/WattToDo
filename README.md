# 🚗⚡ WattToDo — 충전 시간을 가치 있게 만드는 EV 라이프 플랫폼

> 전기차 충전 과정에서 발생하는 ‘대기 시간’을 단순한 기다림이 아닌 **의미 있는 활동 시간**으로 전환할 수 있도록 AI 기반 추천과 탄소 절감 시각화를 제공하는 플랫폼입니다.

---

## 📌 1. 프로젝트 개요

### 프로젝트 목표
* **시간 효율 향상:** AI 기반 주변 활동 추천으로 충전 시간을 효율적으로 활용
* **친환경 유도:** 탄소 절감량 시각화와 랭킹 시스템을 통해 지속적인 친환경 행동 유도
* **불편 최소화:** 충전소 정보 제공 및 차량 제원 기반 충전 시간 계산으로 서비스 정확도 향상

### 타겟 사용자
| 사용자 유형 | WattToDo가 제공하는 가치 |
| :--- | :--- |
| 충전 대기 시간이 비효율적인 EV 사용자 | AI가 충전 시간에 맞는 활동을 자동 추천하여 시간 효율 향상 |
| 친환경 실천에 관심 있는 사용자 | 대시보드로 절감량·목표 달성률·순위 즉시 확인 가능 |
| 낯선 지역·도심에서 충전하는 EV 사용자 | 지도 기반 주변 장소 탐색과 이동시간/충전시간 적합도 제공 |

## ✨ 2. 주요 기능 (Core Features)

### 2.1. 전기차 충전소 정보 제공
* **지도 기반 탐색:** 공공데이터 및 카카오맵 연동으로 충전소 정보 표시
* **필터링:** 충전기 타입, 개방 여부, 속도, 충전기 개수 등으로 상세 필터링 기능 제공
* **차량 제원 기반 충전 시간 계산:** 차량 정보(배터리 용량, 현재 충전량, 목표 충전량, 충전 속도)를 활용하여 예상 충전 시간($T$)을 정확하게 계산합니다.

### 2.2. AI 기반 주변 장소 추천 (타임킬링)
사용자 현재 위치, **예상 충전 시간**, 선호 키워드, 간단한 날씨 정보를 기반으로 충전 시간 내 가능한 활동을 추천합니다.
* **추천 로직:** 카카오맵 API로 주변 장소(POI) 수집 후, 충전 시간 적합도에 따라 1차 필터링합니다. 이후 LLM(생성형 AI)이 맥락을 평가하고 **추천 이유를 자연어로 생성**하여 제공합니다.
* **핵심 정보:** 장소명, 이동 시간/거리, 예상 체류 시간, **충전 시간 적합도** (High/Medium/Low)

### 2.3. 탄소 절감 대시보드 및 랭킹 시스템
친환경 실천 결과를 직관적으로 시각화하고 사용자 참여를 유도합니다.
* **대시보드:** 누적 $\text{CO}_2$ 절감량, 절감 추이 그래프(월간/연간) 등 **KPI 카드형 위젯**으로 제공합니다.
* **랭킹 시스템:** 충전 이벤트 기반 절감량 누적치로 사용자 간 **실시간 순위**를 비교하여 지속적인 참여를 유도합니다.

### 2.4. 회원 및 차량 관리
* **회원 기능:** 회원가입, 로그인, 비밀번호 찾기, 사용자 정보 수정 등
* **차량 관리:** 차종, 충전 타입, 배터리 용량, 최대 주행거리, 전비(전기차 연비) 등 상세 제원 관리 기능 제공

| 구분 | 기술 스택 | 비고 |
