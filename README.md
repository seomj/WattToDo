# 🚗⚡ WattToDo — 충전 시간을 가치 있게 만드는 친환경 EV 플랫폼

> 전기차 충전 시간 동안의 **비효율적 대기 문제**를 해결하고, **AI 기반 시간 활용 가이드**와 **탄소 절감 시각화**를 제공하여 지속 가능한 친환경 행동을 촉진하는 플랫폼입니다.

---

## 📌 1. 프로젝트 개요

* **프로젝트명:** WattToDo (왓투두)
* **목적:**
    * 전기차 충전 대기 시간의 비효율성 해소 및 가치 있는 시간 활용 유도
    * AI 추천을 통한 주변 활동 가이드 제공
    * 탄소 절감 데이터 시각화 및 친환경 랭킹 시스템을 통한 사용자 참여 촉진
* **타겟 사용자:** 충전 대기 시간이 길어 비효율을 느끼는 **전기차 사용자**, 탄소 절감 결과를 시각적으로 확인하고 싶은 **친환경 관심층**, **도심 충전 사용자**

---

## ✨ 2. 주요 기능

### 1) 전기차 충전소 정보 제공 (메인 기능)

* **지도 기반 탐색:** 공공데이터 기반 충전소 정보를 카카오맵과 연동하여 제공
* **필터링:** 충전기 타입, 개방 여부, 충전 속도 등으로 필터링
* **추천:** 사용자 위치 기반 충전소 추천
* **관리:** 즐겨찾기 등록 기능 제공

### 2) AI 기반 주변 장소 추천

* **추천 로직:** 사용자 위치, 시간대, **예상 충전 시간**을 기반으로 주변 장소 추천
* **데이터 소스:** 카카오맵 장소 검색 API 및 LLM(생성형 AI) 활용
* **추천 카테고리:** 카페, 식당, 산책로, 제로웨이스트샵, 편의시설 등
* **결과 제공:** 장소명, 거리, 소요 시간, 평점, **충전 시간 동안 가능한 활동** 등을 계산하여 추천 이유와 함께 제공

### 3) 탄소 절감 대시보드 + 친환경 랭킹 시스템

* **CO₂ 절감량 산정:** 차량 연비, 주행거리, 충전량을 기반으로 **CO₂ 절감량 자동 산정**
    * *(계산 참고식 포함 예정: 휘발유/경유 1L당 탄소 배출량 및 EV 충전 시 배출량)*
* **시각화:** 월별 절감량 그래프 및 누적 절감량 확인
* **경쟁 요소:** 사용자 간 **친환경 랭킹** 제공

### 4) 회원 기능

* **기본 기능:** 회원가입, 로그인, 로그아웃, 정보 수정, 차량 정보 관리 (차종, 충전 타입)
* **기록 관리:** 충전 기록 확인 및 즐겨찾기 목록 관리
* **통합 관리:** 충전 기록 및 탄소 절감량 통합 관리 기능
