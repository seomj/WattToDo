# 🚗⚡ WattToDo — 충전 시간을 가치 있게 만드는 EV 라이프 플랫폼

> 전기차 충전 과정에서 발생하는 ‘대기 시간’을 단순한 기다림이 아닌 **의미 있는 활동 시간**으로 전환할 수 있도록 AI 기반 추천과 탄소 절감 시각화를 제공하는 지능형 플랫폼입니다.

---

## 📌 1. 프로젝트 개요

### 1.1 프로젝트 배경
전기차 보급 확대에도 불구하고 사용자는 여전히 **충전소 정보의 파편화**, **대기 시간의 비효율성**, **환경 기여도 체감 부족**이라는 불편을 겪고 있습니다. WattToDo는 이를 해결하기 위해 충전 시작부터 기록 저장까지의 스마트 프로세스를 구축합니다.

### 1.2 프로젝트 목표
* **실시간 인프라 정보:** 한국환경공단에서 제공하는 데이터 API를 통한 직관적인 충전소 상태 제공
* **AI 맞춤형 큐레이션:** 예상 충전 시간 및 상황(날씨, 취향)을 반영한 활동 추천
* **친환경 가치 시각화:** 내연기관 대비 탄소 절감 효과를 데이터로 환산 및 위젯 제공
* **스마트 프로세스:** 위치 기반 충전 시작 인증 및 AI OCR을 활용한 자동 충전 기록 관리

---

## ✨ 2. 주요 기능 (Core Features)

### 2.1. 지능형 전기차 충전소 탐색
* **Kakao Map API 연동:** 사용자 위치 기반 주변 충전소 실시간 조회 및 필터링
* **충전 시뮬레이션:** 차량 제원(배터리 용량, 잔량)과 충전기 출력을 결합하여 예상 소요 시간 자동 계산

### 2.2. AI 기반 상황 인지 추천
* **API 오케스트레이션:** Kakao(장소), Naver Clova(분석), OpenAI(GMS)를 결합한 데이터 파이프라인 구축
* **맥락 인식 알고리즘:** 충전 시간, 날씨, 개인 선호도 등을 고려하여 LLM이 "해당 장소가 적합한 이유"를 자연어로 설명

### 2.3. 스마트 충전 및 기록 관리
* **위치 인증 기반 충전:** 충전소 반경 30m 이내일 때만 시작을 허용하여 데이터 부정확성 방지
* **AI OCR 기록 자동화:** 충전 완료 문자 이미지를 업로드하면 충전량, 금액, 시간을 자동 추출 및 저장
* **탄소 절감 대시보드:** 누적 $CO_2$ 절감량을 시각화하여 환경 기여도 제공
